# Logistics Website

This is a static, host-ready logistics company homepage inspired by the reference screenshot.

## Files

- `index.html` - page content and structure
- `styles.css` - visual design and responsive layout
- `script.js` - mobile menu and current year

## Customize

1. Replace the sample company name and logo text in `index.html`.
2. Update phone numbers, email address, WhatsApp link, and form action.
3. Edit service names, stats, and section copy.
4. Replace the background image URLs in `styles.css` if you want your own terminal or fleet photos.

## Host Live

Upload the full folder to a static host such as Netlify, Vercel, GitHub Pages, or any web hosting control panel.

For Netlify, visit <https://app.netlify.com/drop> and drag this folder into the upload area.

## Local Preview

Run this from the folder:

```bash
python -m http.server 4173
```

Then open:

```text
http://localhost:4173
```

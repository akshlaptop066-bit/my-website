# Enterprise Logistics Website

This is a static, host-ready enterprise logistics and infrastructure website.

## Files

- `index.html` - home page
- `about.html` - company profile
- `services.html` - service catalog
- `infrastructure.html` - fleet and infrastructure
- `projects.html` - project showcase
- `gallery.html` - image gallery with lightbox
- `careers.html` - careers page
- `contact.html` - contact and quote page
- `styles.css` - visual design and responsive layout
- `script.js` - loading screen, dark mode, counters, reveal animations, sliders, lightbox, and page transitions

## Customize

1. Replace the sample company name and logo text in `index.html`.
2. Update phone numbers, email address, WhatsApp link, and form action.
3. Edit service names, stats, projects, gallery images, and section copy.
4. Replace the image URLs in the HTML/CSS if you want your own terminal or fleet photos.

## Host Live

Upload the full folder to a static host such as Netlify, Vercel, GitHub Pages, or any web hosting control panel.

For Netlify, visit <https://app.netlify.com/drop> and drag this folder into the upload area.

## Local Preview

Run this from the folder:

```bash
python -m http.server 4173
```

Then open:

```text
http://localhost:4173
```
